## Description

Ce TD porte sur les **réseaux sémantiques**, qui sont des représentations graphiques de connaissances permettant de structurer des informations sous forme de concepts et de relations.  
L'objectif est de modéliser des connaissances dans différents domaines (ex : médical, animalier) et de manipuler ces réseaux pour en déduire des informations ou des relations implicites.
